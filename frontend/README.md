# Local Travel Experience Frontend

This is the React frontend for the Local Travel Experience application.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

## Features

- **Authentication**: Login and Register (JWT based).
- **Home Feed**: View recent and trending posts (Food, Concerts, Events).
- **Create Experience**: Share your own local experiences.
- **Post Details**: View full details, images, and comments.
- **Social**: Like, Upvote, and Comment on posts.
- **Profile**: View your profile and shared posts.

## Tech Stack

- React (Vite)
- React Router DOM
- Axios
- Lucide React (Icons)
- Vanilla CSS (Premium Dark Theme)
